How it works

A tiny app with
a very firm opinion.

01

Choose distractions

Add individual domains or use presets for News, Social Media, and Entertainment.

02

Start strict mode

Choose how many random words you must type before Hotblock lets you stop.

03

Try to procrastinate

Hotblock closes the tab and delivers an increasingly direct spoken reminder.

“You chose to focus.
Not visit instagram dot com.”

Warnings get harsher when you keep trying.

Not another silent blocker

It talks back.

Pick an English macOS voice and volume. Every repeated attempt raises the shared warning level until you finally get the hint.

Install Hotblock

From download
to focus in minutes.

  1. 1

    Download Hotblock.zip

    Get the latest developer preview from GitHub Releases.

  2. 2

    Move Hotblock to Applications

    Unzip the download, then drag Hotblock.app into your Applications folder.

  3. 3

    Right-click and choose Open

    The preview is not Apple-notarized yet, so macOS may ask you to confirm.

  4. 4

    Allow browser permissions

    Follow Hotblock's setup assistant to enable Automation and notifications.
