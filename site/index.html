Install from GitHub

Free, open,
and built in public.

Every release is built from the public source by GitHub Actions. Hotblock is not Apple-notarized yet, so macOS asks you to approve it once.

  1. 1

    Download Hotblock.dmg

    Get the latest Mac disk image from GitHub Releases.

  2. 2

    Move Hotblock to Applications

    Open the disk image, then drag Hotblock into your Applications folder.

  3. 3

    Approve the GitHub beta once

    Right-click Hotblock and choose Open. If blocked, use System Settings → Privacy & Security → Open Anyway.

  4. 4

    Allow browser permissions

    Enable Automation for your browsers. Notifications are optional.
